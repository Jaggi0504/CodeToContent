**Automate Your Notebook Docs – From Code to README & LinkedIn in One Click**  

Data teams spend countless hours turning exploratory notebooks into polished documentation and social‑media updates. This repo eliminates that manual step with an end‑to‑end pipeline that reads any Jupyter `.ipynb`, extracts markdown and code cells, and uses **LangChain**, **LangGraph**, and **Groq’s `gpt‑oss‑120b`** LLM to generate a production‑ready GitHub README *and* a concise LinkedIn post in seconds.  

**Key tools & frameworks**  
- **nbformat** – reliable notebook parsing  
- **PromptTemplate** (LangChain) – custom prompts for README & LinkedIn copy  
- **ChatGroq** – deterministic LLM output (temp 0.3)  
- **langgraph.StateGraph** – deterministic workflow (generate README → generate LinkedIn)  
- **Python ≥ 3.9**, **dotenv**, **IPython.display** (Mermaid visualisation)  

Why it matters:  
- Keeps documentation in sync with the source notebook, reducing drift.  
- Saves engineers ≈ 30 min per notebook on copy‑editing and formatting.  
- Produces output that speaks directly to analysts, data scientists, ML engineers, and AI developers—technical yet natural.
